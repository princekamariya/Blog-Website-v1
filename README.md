This is Blog Website version 1. This project is a mini version of a working blogging website. This project is done by using Node.JS, EJS, CSS, HTMl. Here Anyone can write on Daily Blogs. ## Tech Stack

**Client:** NodeJS, EJS, HTML, CSS, Bootstrap



## Run Locally

Clone the project

```bash
  git clone https://github.com/prins2003/Daily-Blogs.git
```

Go to the project directory

```bash
  cd Daily-Blogs
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  node app.js
```
